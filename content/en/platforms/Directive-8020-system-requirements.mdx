## Best Settings Strategy for Minimum and Mid-Range PCs

If your rig only slightly exceeds the Directive 8020 system requirements, smart settings matter more than brute force.

### High-impact settings to tune first

| Setting | Performance Cost | Visual Trade-Off | Recommended Starting Point |
|---|---|---|---|
| **Shadows** | High | Moderate in dark scenes | **Medium** |
| **Ray Tracing** | Very High | Noticeable reflections/lighting | **Off** (or Low RT if stable) |
| **Post-Processing** | Medium | Filmic effects, bloom | **Medium** |
| **Texture Quality** | VRAM dependent | Sharpness/detail | **High if VRAM allows** |
| **Volumetrics/Fog** | Medium to High | Atmosphere depth | **Medium** |
| **Path Tracing** | Extreme | Premium lighting realism | **Off on mid-range GPUs** |

### Quick optimization sequence

1. Set native resolution first (1080p or 1440p).
2. Disable path tracing before touching texture quality.
3. Lower shadows from High to Medium.
4. Use available upscaling (DLSS/FSR/XeSS style options if exposed).
5. Cap FPS (for example 60) to improve frame pacing.
6. Test during a dense interior scene, not just menus.

This sequence keeps visual identity intact while reducing common frame dips.

## Pre-Launch and Day-One Checklist for Smooth Play

Meeting the Directive 8020 system requirements is step one. Clean setup discipline is step two.

### Pre-launch readiness table

| Task | Why It Matters | When to Do It |
|---|---|---|
| **Update GPU driver** | Fixes optimization and stability issues | 1–2 days before launch |
| **Install on SSD** | Faster loads, fewer streaming hitches | During install |
| **Verify 40+ GB free space** | Patches and temp files need margin | Before preload |
| **Close overlay-heavy apps** | Reduces CPU/RAM conflicts | Before first boot |
| **Set power mode to High Performance** | Improves boost behavior | Before benchmarking |
| **Cap FPS if unstable** | Better frame pacing in cinematic scenes | After first test run |

### Recommended launch-day workflow

1. Boot the game with default preset.
2. Run one intense scene for 5–10 minutes.
3. Watch frame-time consistency, not only average FPS.
4. Adjust two settings at a time, then retest.
5. Keep a fallback “stable” preset profile.

> **⚠️ Warning:** Avoid changing every graphics option at once. You won’t know which setting fixed (or caused) instability.
