## なぜこのキャストは過去のDark Pictures作品と違って感じられるのか

Supermassiveは、より現実的なプロフェッショナル設定を用意したうえで、そこにパラノイアをかけて負荷試験しているように見えます。カジュアルな人間関係の集団よりもドラマのコントラストが強くなるのは、各人が最初はプロトコル、役割の明確さ、任務責任を持っているのに、そこから安定要素を失っていくからです。

### 主要な設計上の違い

| Element | Earlier Anthology Pattern | Directive 8020 Direction |
|---|---|---|
| グループ構造 | 緩くつながった個人であることが多い | **事前選抜された宇宙飛行士チーム** |
| 対立の引き金 | 個人的衝突 + 超常的脅威 | **任務プレッシャー + 身元偽装** |
| キャラクターの賭け金 | 主に個人の生存 | **個人生存 + 種レベルの危機** |
| 役割機能 | 性格主導のシーン | **性格 + 技術的責任** |

この転換は、物語テンポを大きく改善する可能性があります。「この人たちは誰？」を長く引っ張る代わりに、プレイヤーはより早く意味のあるジレンマへ到達できるかもしれません。たとえば指揮系統の破綻、研究倫理、隔離判断、そして犠牲の論理です。

## キャストに関する選択がエンディングとリプレイ価値にどう影響しそうか

新機能のTurning Points（主要な物語ノードへ巻き戻し）から、Supermassiveがプレイヤーにキャスト結果の再検証を何度も行わせる設計を想定していることが分かります。実際的には、**Directive 8020 cast**は単純な「生存/死亡」の二択ではなく、重層的な結果分岐で構築されている可能性が高いということです。

### 注目すべき意思決定プレッシャーポイント

1. **身元確認シーン**  
   慎重さ（遅いが安全）と緊急性（速いが危険）のどちらを取るかが問われます。

2. **指揮系統の上書き判断**  
   権威を支持するか、逆らうかで、クルー結束や資源アクセスが変化する可能性があります。

3. **犠牲の計算**  
   個人コストを払って任務成功を守るか、優先順位を反転させるかの選択が発生します。

4. **情報統制**  
   汚染リスクを誰がいつ知るかが、連鎖反応を左右しうる要素になります。

| Choice Type | Short-Term Result | Long-Term Cast Consequence |
|---|---|---|
| 見知ったクルーを即座に信じる | 進行が速まる | 偽者を通すリスク上昇 |
| 証拠/プロトコル確認を要求する | 遅延と対立が発生 | 確実性は上がるが士気は低下 |
| 脅威情報を秘匿する | 目先のパニック回避 | 後の裏切り反発を招く |
| 脅威情報を全面共有する | チーム全体の準備性向上 | 即時の人間関係分断 |

「全員生存」ルートを狙うプレイヤーにとって、リプレイ機能はほぼ必須になるでしょう。本作はQTEのタイミング暗記よりも、キャスト心理の理解のほうが価値を持つタイプのゲームになりそうです。
