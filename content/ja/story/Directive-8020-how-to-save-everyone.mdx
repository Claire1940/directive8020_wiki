## 10名の宇宙飛行士クルー：誰が生き残れるか？

本作における最も大きな変化は、クルーの人数です。過去の作品では通常5人の主人公を管理していましたが、本作では10名の宇宙飛行士に対して責任を負うことになります。一部は主要なプレイアブルキャラクターですが、本作では「非操作キャラクター（NPC）」の生存という、より深いレイヤーが導入されています。最良の結果を得るためには、一人ひとりをミッションに不可欠な資産として扱う必要があります。

クルーの生存は相互に関連しています。ゲーム序盤で専門の技術者を失うと、後半のキャラクターによる修理シーケンスが大幅に難化する可能性があります。クルーの役割を理解することが、**Directive 8020 how to save everyone**戦略を立てる第一歩です。

| クルーの役割 | 主な責任 | 生存難易度 |
| :--- | :--- | :--- |
| **指揮官** | 意思決定と士気管理 | **高**（序盤に狙われやすい） |
| **エンジニア** | 船体の修理と技術的バイパス | **中**（終盤に必須） |
| **医療官** | 負傷の治療と感染スキャン | **致命的**（永久的な損失を防ぐ） |
| **セキュリティ/戦術** | 戦闘遭遇と護衛 | **中**（犠牲になるリスクが高い） |
| **科学リーダー** | 擬態（ミミック）の脅威の特定 | **高**（研究中に脆弱になる） |

> **💡 ヒント:** 非操作クルーを軽視しないでください。今回初めて、彼らの生存が最終ランクに影響し、以前は利用できなかったユニークなエンディング映像をアンロックできるようになりました。

## 「デススパイラル」メカニクスの理解

ゲームプレイのループに追加された最も危険な要素の一つが「デススパイラル（死の連鎖）」です。これは、物語の中盤あたりで発生する隠れたメカニクスです。誤った決断を重ねすぎたり、主要な人員を失ったりすると、ゲームはパーティーの全滅がほぼ避けられない状態に陥ることがあります。

デススパイラルを回避し、**Directive 8020 how to save everyone**を達成するためには、成功した選択の「バッファー」を維持し続ける必要があります。デススパイラルは累積的なペナルティシステムとして機能します。船体の整合性が低すぎたり、クルーの信頼が崩壊していたりすると、たった一つのイベントの失敗が、残りの全キャラクターが次々と排除されるドミノ倒しのような効果を引き起こします。

| 特徴 | 生存への影響 | 軽減方法 |
| :--- | :--- | :--- |
| **クルーの信頼** | キャラクター間の協力に影響 | ダイアログでの透明性のある対話 |
| **船体の整合性** | 環境ハザードの発生率を決定 | 探索よりも修理タスクを優先する |
| **感染レベル** | キャラクターの突然の変異を引き起こす | 定期的な健康診断とスキャン |
| **リソース管理** | 危機時の選択肢を制限する | 進行前に全セクターを探索して回収する |

## リアルタイム・サバイバルとステルス

純粋にシネマティックだった過去作とは異なり、本作はリアルタイムのステルスと戦闘メカニクスを取り入れています。**Directive 8020 how to save everyone**を知りたいのであれば、発見されずに移動する技術をマスターしなければなりません。タウ・セティfのエイリアンの脅威は「擬態」を利用します。つまり、クルーの誰か、あるいは無機物でさえも脅威である可能性があるのです。

1.  **スキャナーを頻繁に使用する:** 手持ちのスキャナーは生物学的異常を検知できます。新しい部屋に入る前に使用して、擬態を特定してください。
2.  **息を止める:** 「Don't Breathe（息を止めるな）」メカニクスが復活しましたが、今回はリアルタイムの隠れ場所と統合されています。UIを注視して冷静さを保ちましょう。
3.  **環境を利用した陽動:** 船のシステムを使って遠くの部屋で音を出し、脅威を自分の進路から遠ざけます。
4.  **戦闘は最終手段:** 直接戦闘に突入することは、キャラクターを失う最短の道です。常に脱出ルートやステルスによる回避策を優先してください。

## 非操作キャラクター（NPC）を救う

2026年版のゲームプレイモデルにおける大きな進化は、間接的な行動を通じてNPCを生存させられるようになったことです。これまでのタイトルでは、NPCは脚本によって「運命づけられている」ことが多かったのですが、開発者は背景のクルーを安全に保つための「非常に興味深い方法」があると明言しています。

これらのキャラクターを確実に生存させるには、あえて困難な道を選ぶ必要があることが多々あります。例えば、サポートスタッフのために通路を確保するために危険なルートを選んだり、限られたリソースを消費して、必須ではない区画の生命維持装置を修理したりすることです。これらの行動は操作キャラクターに直接的な利益をもたらさないこともありますが、「パーフェクト・サバイバル」のトロフィーを狙うなら必須です。

> **警告:** 本作では、過去の作品よりもはるかに早い段階でキャラクターが死亡する可能性があります。最初の30分間でのミスが、フルクルー生存のランを永久に不可能にする場合があります。

## 2026年版の重要な決断ポイント

すべての選択が重要ですが、中にはより重みを持つものがあります。**Directive 8020 how to save everyone**を目指す際は、「モラルコンパス」のプロンプトに細心の注意を払ってください。これらはしばしばキャラクター間の長期的な関係を左右します。二人のキャラクターが互いを嫌悪している場合、ゲーム後半のスクリプト化されたQTE（クイックタイムイベント）で互いを助ける可能性が低くなります。

| 決断のタイプ | 即時効果 | 長期的な生存への影響 |
| :--- | :--- | :--- |
| **無私の行動** | キャラクターがダメージ/リスクを負う | 信頼度が上昇し、「救出」プロンプトが有効になる |
| **現実的な選択** | キャラクターの安全を確保する | 士気が低下し、デススパイラルを誘発する可能性がある |
| **正直さ** | 対立の可能性 | クルー内の擬態を特定するために不可欠 |
| **攻撃性** | 目前の脅威を取り除く | しばしばクルーの誤射や不慮の事故を招く |

ゲームの開発や世界観に関する詳細については、[Supermassive Gamesの公式サイト](https://www.supermassivegames.com)を訪問し、最新のパッチやストーリー拡張の情報を確認してください。

## 擬態（ミミック）の脅威の管理

本作の核心的なホラー要素は「擬態」です。エイリアンの有機体はクルーの姿を模倣することができます。これにより、サバイバルゲームプレイに「正体隠匿（ソーシャル・ディダクション）」のレイヤーが加わります。誤って本物のクルーではなく擬態を助けてしまった場合、それは他の宇宙飛行士の死に直結します。

**Directive 8020 how to save everyone**を完璧に遂行するには、船内の至る所にある「検証プロトコル（Verification Protocols）」を使用する必要があります。これらは使用回数が限られたアイテムやステーションで、キャラクターの正体を確認できます。これらをクルーが分断されている緊張感の高い瞬間のために温存しておくことが、内部からの大虐殺を防ぐ最も効果的な方法です。

## 生存ステップのまとめ

まとめとして、パーフェクトなランを達成するには、伝統的な意思決定と新しい技術的なスキルの融合が必要です。
*   主人公だけでなくNPCの安全も優先し、10名のクルーを維持する。
*   船体の整合性とクルーの信頼を高く保ち、「デススパイラル」を回避する。
*   序盤の死亡を防ぐためにリアルタイム・ステルスをマスターする。
*   スキャナーと検証プロトコルを使い、攻撃される前に擬態を特定する。
*   キャラクターが予想よりもずっと早い段階で致命的な危険にさらされることを覚悟しておく。

## FAQ

**Q: Directive 8020で本当に10名全員を救うことはできますか？**

A: はい、非操作キャラクターを含む10名全員の宇宙飛行士を救うことは可能です。ただし、それには「デススパイラル」メカニクスや隠されたNPC救出目標に細心の注意を払ったパーフェクトなプレイが必要です。

**Q: 「デススパイラル」とは何ですか？どうすれば回避できますか？**

A: デススパイラルは、ゲームの中盤あたりで発生する隠れた累積的な失敗状態です。**Directive 8020 how to save everyone**を模索する過程でこれを避けるには、物語の前半で一貫してQTEに成功し、船体の修理を維持し、クルーの士気を高く保つ必要があります。

**Q: NPCは「全員生存」の実績に含まれますか？**

A: 本作では、NPCはかつてないほど重要な存在です。主要な実績は通常プレイアブルキャラクターに焦点を当てていますが、追加のクルーを救うことで「真の」ベストエンディングがアンロックされ、物語により深い背景が与えられます。

**Q: クルーが殺される前に擬態を見分ける方法はありますか？**

A: はい、プレイヤーはカシオペア号内にある生物学的スキャナーや検証ステーションを使用できます。また、会話中の微妙な行動の変化に注意を払うことで、キャラクターが擬態に入れ替わっているかどうかのヒントを得ることができます。
